## Base URLs

| Environment | URL |
| --- | --- |
| Production | `https://api.feedling.app` |
| Staging | `https://test-api.feedling.app` |

All API requests and responses use JSON unless an endpoint explicitly documents
file upload or binary content.

The API reference is the field-level HTTP contract. The workflow guides explain
how those operations fit together and call out capabilities that the service does
not currently provide.
